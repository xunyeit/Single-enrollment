### 一、产品概述
**“****新昇学****”是一款专为****中职生、高中生及社会考生群体****打造的****单招考试（单独招生）信息服务平台****，聚焦于职业教育与普通高校衔接的升学路径。平台以“****助力学子圆梦大学，精准把握升学机会****”为核心使命，整合全国范围内高职院校的单招政策、招生章程、录取分数、备考资料等权威信息，提供一站式查询、学习、交流与报名服务。**

**该应用面向有意向通过“单独招生”方式进入高等职业院校的学生及其家长，覆盖政策解读、院校对比、题库练习、在线答疑、报名记录管理等功能，帮助用户全面掌握单招流程，科学规划备考策略，提升升学成功率。**

**“新升学”不仅是信息聚合工具，更是连接学生与优质教育机会的桥梁，推动职业教育公平化、透明化发展，助力更多青年实现人生跃迁。**

### 二、核心功能
1. **单招政策****  
****实时更新国家及地方各级教育主管部门发布的单招政策文件，包括报名条件、考试时间、录取规则等，确保信息权威准确。**
2. **单招院校****  
****提供全国各省份参与单招的高职院校列表，支持按地区、专业、层次筛选，展示学校简介、师资力量、就业率等关键信息。**
3. **单招章程****  
****汇集各院校发布的官方招生章程，内容涵盖专业设置、招生计划、考试科目、评分标准、录取原则等，便于学生深入研究。**
4. **资料下载****  
****支持免费下载历年真题、模拟试题、复习讲义、报考指南等学习资料，助力高效备考。**
5. **录取分数****  
****展示往届各院校、各专业的录取分数线，支持按年份、省份、专业排序查看，帮助学生科学估分择校。**
6. **荣誉榜****  
****公布优秀学员案例、高分考生名单、成功上岸故事，激励用户树立信心，营造积极向上的学习氛围。**
7. **在线答疑****  
****提供即时咨询服务，用户可提交关于政策、报名、考试等问题，由专业老师或客服团队及时解答。**
8. **交流群****  
****进入专属学习社群，与同地区、同目标的考生互动交流经验，分享备考心得，形成互助学习圈。**
9. **我的钱包****  
****管理账户余额，用于购买会员服务、付费课程、资料包等增值服务。**
10. **购买会员****  
****提供VIP会员订阅服务，享受优先答疑、专属题库、视频课程、一对一指导等高级权益。**
11. **我的团队****  
****支持教师或机构用户创建教学团队，管理学员账号、发布通知、布置任务，适用于培训机构或辅导班使用。**
12. **联系我们****  
****提供客服电话、邮箱、在线客服入口，方便用户反馈问题或寻求帮助。**
13. **关于我们****  
****展示平台运营主体、服务理念、合作单位、发展历程等信息，增强品牌信任感。**
14. **报名记录****  
****查看已提交的单招报名信息，包括报考院校、专业、状态（待审核、已录取等），便于跟踪进度。**
15. **清除缓存****  
****手动清理本地数据缓存，优化App运行速度，保护隐私安全。**
16. **单招题库****  
****（主界面功能）内置海量历年真题与模拟试题，支持按科目、难度、题型分类练习，并提供智能错题本与解析。**
17. **提分秘籍领取****  
****（主界面活动）推出限时福利活动，如“领取提分秘籍”“免费试听课程”，吸引用户参与并提升活跃度。**

### 三、产品价值
**用户价值：让升学更透明、更高效、更有方向**

+ **信息全面透明****：打破信息不对称，将复杂的单招政策、院校信息、录取数据集中呈现，帮助学生做出理性选择。**
+ **备考更有针对性****：通过资料下载、题库练习、真题分析等功能，提升应试能力，减少盲目刷题。**
+ **决策更科学****：结合录取分数、院校排名、专业热度等数据，辅助学生合理定位目标院校，避免“高分低报”或“滑档”风险。**
+ **学习不孤单****：加入交流群、参与荣誉榜，获得同伴支持与榜样激励，增强学习动力。**
+ **全程可追踪：从政策了解、资料准备、报名提交到结果查询，全流程可视化管理，减轻焦虑感。**

## 架构特点
### 架构特性
+ **前后端分离架构，独立开发，符合主流开发模式**
+ **前端以Vue3+Vite为主技术，AntdV为UI界面框架**
+ **后端SpringBoot3为基础，MybatisPlus为数据操作框架，Redis为缓存框架**
+ **Maven多模块管理，插件化开发，方便安装、卸载、升级，降低耦合**
+ **业务模块与API抽离，模块之间便捷引用**
+ **数据库设计精巧，字段规范、易于扩展**
+ **支持国产密码算法加解密，等保测评国产项目无压力**
+ **支持MYSQL、ORACLE、SQLSERVER、PGSQL等主流标准结构式数据库**
+ **支持达梦、人大金仓、南大通用、九有、瀚高、虚谷数据库等国产数据库**
+ **支持中创、宝蓝德、东方通等中间件**
+ **支持Windows、Linux操作系统、国产操作系统部署**

### 功能特性
+ **完善的系统基础功能，满足使用需求，避免重复造轮子**
+ **支持本地文件、阿里云文件、腾讯云文件、MINIO文件上传**
+ **支持本地邮件、阿里云邮件、腾讯云邮件发送**
+ **支持阿里云短信、腾讯云短信发送**
+ **B、C端双账号认证体系，会话治理各自独立**
+ **完善的登录日志、操作日志、异常日志等审计功能**
+ **完善的会话监控、数据源监控、系统监控等必备监控功能**
+ **支持组织机构、权限管理、定时任务、系统配置等基础功能**

### 安全特性
+ **采用SaToken轻量级 Java 权限认证框架，功能强大、学习成本低**
+ **支持登录认证、权限认证、单点登录、三方登录、OAuth2.0等认证模式**
+ **增强的RBAC权限设计，资源于接口独立授权，更加灵活**
+ **支持按钮级别细粒度独立授权，界面按钮动态展示**
+ **支持API接口注解式、路由拦截式鉴权，防止越界访问**
+ **独创的数据范围机制，每个接口都可以配置不同数据范围**
+ **支持限流防抖，防重复提交，有效阻止脏数据产生**
+ **密码、手机、身份证号等使用国密算法加密传输、加密存储，数据更安全**
+ **操作日志使用SM2进行完整性保护，满足安全审计要求**

### 界面特性
+ **Vue3 + Vite为基础，AntdV为界面UI框架，视觉风格清新简洁**
+ **精细化设计，注重界面的每一处细节，操作轻松友好**
+ **暗黑风格、经典菜单、双排菜单、多页签、目录坞、主题切换等功能应有尽有**
+ **统一的网络框架、API接口拦截框架，拿来即可上手**

## 运维环境
+ **全系列系统服务器。推荐使用 Centos 7.6**
+ **2 核 2 G 100MB 3M带宽 为最低配置，建议 2 核 4G 100Mb 5M带宽。**
+ **Node.js 14x**
+ **Java1.8**
+ **Mysql 5.7**
+ **Redis 6.2.8**
+ **阿里云 OSS – 可更换其它。**
